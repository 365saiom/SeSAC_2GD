Q1. Javascript에서 함수를 만드는 방법으로 옳은 것은?

  1. function = myFunction()
  2. function myFunction()
  3. function:myFunction()
  4. def myFunction():

(해설) 자바스크립트에서 함수를 선언하는 방식은 명시적 함수 선언과 함수 표현식 두 가지가 있다.
명시적 함수 선언 방식으로는 function myFunction() 다음과 같이 함수를 정의하며, 함수 표현식으로는 const myFunction = function () {} 다음과 같이 정의할 수 있다.

​

Q2. Javascript에서 myFunction 함수를 호출하는 방법으로 옳은 것은?

  1. call function MyFunction()
  2. call myFunction()
  3. myFunction()

(해설) 자바스크립트에서 함수를 호출할 때 함수 이름과 괄호를 이용해서 호출한다.

​

Q3. 다음 중 결과가 다른 하나는?

  1. !true
  2. true && false
  3. true || false
  4. !!false && (3 > 5)

(해설) 1, 2, 4는 false 3번은 true값이다.

​

Q4. 다음 중 결과가 다른 하나는?

  1. 3 !== '3'
  2. (8 / '2') !== 4
  3. 0 === false
  4. ('1' + 2) === 12

(해설) ==, != 연산자는 두 피연산자의 타입이 다르더라도 값만 비교하지만
===, !== 연산자는 엄격한 동등 연산자로 피연산자의 값과 타입을 모두 비교한다.

​

Q5. 다음 중 if 문을 올바르게 작성한 것은 무엇인가?

  1. if (x == 5)
  2. if (x == 5):
  3. if (x == 5) then
  4. if x = 5

(해설) 조건문은 if (조건식)으로 작성한다.

Q6. 자바스크립트에서 "i가 7이 아닐 때"를 의미하는 코드로 올바른 것은?

  1. if (i ​ <> 7)
  2. if (i !== 7)
  3. if (i =! 7) then
  4. if i != 7:

(해설) i가 7이 아닐 때이므로 동등 연산자 중에서 != 혹은 !== 연산자를 이용해야 하며, if (조건식) 형식으로 작성해주어야 한다.

Q. 다음 중 삼항 연산자를 이용해 gender가 0이면 '남자', 0이 아니면 '여자'를 출력하는 코드로 옳은 것은? (gender 변수는 이미 선언되어 있다고 가정)

  1. console.log(gender == 0 : '남자' ? '여자');
  2. console.log(gender == 0 ? '남자' : '여자');
  3. console.log(gender == 0 : '여자' ? '남자');
  4. console.log(gender == 0 ? '여자' : '남자');

(해설) 자바스크립트에서 ​ <> 연산자는 존재하지 않으며, while 반복문은 while (조건식) 형식으로 작성해주어야 한다.
